## 🔧 Troubleshooting rapide

| Problème | Solution |
|----------|----------|
| QR code ne scanne pas | Augmenter la taille, enlever les reflets |
| Pas de connexion wifi | Utiliser le hotspot d'un téléphone |
| Formulaire ne charge pas | Vérifier la connexion, recharger la page |
| Feedback ne va pas sur Discord | Vérifier le webhook dans la console (F12) |
| Trop lent sur mobile | Attendre quelques secondes, Discord a un rate limit |
