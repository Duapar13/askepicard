# Askepicard - Feedback JPO

Formulaire de feedback mobile pour la journée portes ouvertes de l'askepicard.

## 📋 Fonctionnalités

- ⭐ Évaluations par étoiles pour :
  - Graphiques
  - Jouabilité  
  - Originalité
- 💬 Champ de suggestions
- 🔗 Envoi automatique via Discord Webhook
- 📱 Design responsive / mobile-first
- 🎨 Interface sombre avec gradient

## 🚀 Utilisation

1. Le formulaire est hébergé sur GitHub Pages
2. Scannez le QR code ou visitez : `https://duapar13.github.io/askepicard/`
3. Remplissez les évaluations et vos suggestions
4. Envoyez : les données seront postées sur Discord automatiquement

## 📝 Données envoyées

Chaque feedback envoie :
- Prénom
- Notes (1-5 étoiles) pour chaque critère
- Suggestions d'amélioration

Les données sont envoyées en tant que webhook Discord avec embed coloré.

## 🔧 Configuration

- **Webhook Discord** : Configuré dans `script.js`
- **Hébergement** : GitHub Pages (branche main)
- **Domaine** : `https://duapar13.github.io/askepicard/`
