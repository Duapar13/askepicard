## 🚀 Comment ça marche

### Pour les visiteurs:
1. Scannez le QR code avec leur téléphone
2. Remplissent le formulaire:
   - ⭐ Évaluations (1-5 étoiles)
   - 💬 Suggestions libres
3. Cliquent sur "Envoyer mon feedback"
4. Les données sont envoyées automatiquement sur Discord

### Pour l'équipe (Discord):
- Tous les feedbacks arrivent automatiquement dans le **serveur Discord**
- Chaque réponse = **un embed coloré** avec les infos

## 📋 Fonctionnalités du formulaire

✅ **Évaluations par étoiles**:
