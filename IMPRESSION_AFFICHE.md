# 📄 Guide Impression Affiche QR Code

## Option 1: Imprimer directement depuis le navigateur (Recommandé ⭐)

1. Ouvrir le fichier `affiche-qr.html` dans Chrome ou Edge
2. **Raccourci**: `Cmd + P` (Mac) ou `Ctrl + P` (Windows)
3. **Paramètres d'impression**:
   - Format: **A4**
   - Orientation: **Portrait**
   - Marges: **Minimum**
   - Arrière-plan: **Activé** ✅
4. **Destination**: Enregistrer en PDF ou Imprimer

## Option 2: Convertir en PDF avec wkhtmltopdf

```bash
# Installer wkhtmltopdf
brew install wkhtmltopdf

# Générer le PDF
bash generate-pdf.sh
```

Le fichier `affiche-askepicard-qr.pdf` sera créé.

## Option 3: Service en ligne gratuit

1. Aller sur **https://cloudconvert.com**
2. Uploads: `affiche-qr.html`
3. Convertir en PDF
4. Télécharger

---

## 🖼️ Recommandations d'impression

| Paramètre | Valeur |
|-----------|--------|
| Format papier | A4 ou A3 (plus grand) |
| Couleur | Oui (le design est en couleur) |
| Résolution | 300 DPI (haute qualité) |
| Marges | Réduites (pour plus de contenus) |
| Orientation | Portrait |
| Qualité | Photo/Meilleure |

---

## 📱 Pour afficher sur écran (JPO)

- Afficher directement le fichier HTML en fullscreen
- Ou utiliser: `https://duapar13.github.io/askepicard/affiche-qr.html`
- Sur un TV/moniteur pour que tout le monde voit le QR code

---

## ✂️ Découpe et affichage

1. **Format recommandé**: A3 ou A4 encadrée
2. **Plastification** (optionnel): Protège l'affiche du temps
3. **Support**: Chevalet photo, cadre, ou scotch sur stand
4. **Hauteur**: À la vue de tous (hauteur des yeux)

---

## 🎯 Dimensions finales

- **A4**: 21 × 29.7 cm
- **A3**: 29.7 × 42 cm (recommandé pour plus de visibilité)
